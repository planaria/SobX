# SobX
[MobX](https://github.com/mobxjs/mobx)-like state management for C++

## Features
- Header-only  
  `#include <sobx/sbox.hpp>`

## Requirements

### Supported Compilers
- Visual Studio 2017

### External Libraries
- [Catch](https://github.com/philsquared/Catch) (for test)
